Класс Game
Инициализирует игровой процесс.
Свойства	Описание
Score	Количество набранных очков
Time	Затраченное время

Методы	                Описание
Initialization	Запускает игровой процесс

Класс Field
Объект данного класса представляет собой игровое поле, состоящий из M?N клеток.
Свойства	                Описание
SizeM	                Кол-во столбцов игрового поля
SizeN	                Кол-во строк игрового поля
Matrix[N][M]	Матрица объектов класса Cell

Методы	Описание
Generate	Генерирует элементы игрового поля
Draw	Рисует игровое поле
Activation	Анализирует возможность активации методов Chain, Squares, Intermix
FirstClick	Анализирует клик по первому элементу игрового поля (квадратику)
SecondClick	Анализирует клик по второму элементу игрового поля (квадратику)
Swap	Перестановка местами двух элементов игрового поля
Chain	Уничтожает цепочки из 3 и более элементов одинаковых по цвету (в том числе Радужного квадрата) по столбцам или строкам игрового поля
Squares	Уничтожает квадраты, которые стоят вокруг элемента (т.е. поле размеров 3х3 клетки) и удаляется сам
Intermix	Перемешивает элементы игрового поля
Scoring	Подсчитывает количество набранных очков

Класс Cell
Объект данного класса представляет собой элемент игрового поля (квадрат).
Свойства	Описание
ObjectType	Тип элемента игрового поля («Базовый» (желтый, красный, синий, зеленый), «Бомба», «Радужный квадрат», «Молния»)

Методы	Описание
CreatElement	Создает элемент игрового поля
RandomElement	Генерирует случайный элемент игрового поля
